# University Timetable Web Application

### Frontend Development Project | HTML, CSS, JavaScript

---

## Overview

The **University Timetable Web Application** is a web-based platform designed to simplify the management and accessibility of university schedules. The system allows students to securely log in and view their academic timetable based on **branch and semester selection**.

This application improves accessibility and organization of academic schedules by providing an intuitive interface that enables quick navigation and easy timetable retrieval.

---

## Key Features

* Secure **Login and Signup functionality**
* **Branch-based timetable filtering**
* **Semester-wise timetable viewing**
* Simple and responsive **user interface**
* Efficient **schedule navigation and accessibility**
* Organized timetable data management

---


---

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Launch **index.html** in any web browser.
4. Create a user account using the signup page.
5. Login and select your **branch and semester** to view the timetable.
